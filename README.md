# 🖱️ Virtual Mouse Using Hand Gesture Recognition

## 📌 Overview
Developed a **Virtual Mouse system** enabling touchless computer interaction using real-time hand gestures captured via a webcam. The project leverages **computer vision and AI-based gesture recognition** to translate hand movements into precise mouse operations, enhancing Human-Computer Interaction (HCI).

## 🚀 Key Highlights
- Implemented real-time hand tracking and gesture recognition  
- Mapped gestures to cursor movement, clicks, scrolling, and system controls  
- Achieved **~97% accuracy** across varying lighting conditions  
- Enhanced usability with smooth and stable cursor control  

## 🛠️ Tech Stack
**Python, OpenCV, MediaPipe, PyAutoGUI**

## 🎯 Outcome
Developed an efficient, gesture-controlled system demonstrating strong expertise in **computer vision, real-time processing, and problem-solving**, with practical applications in accessibility and touchless computing.
